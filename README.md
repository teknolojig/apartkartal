# ApartKartal - Öğrenci Yurdu Web Sitesi

Next.js ile geliştirilen modern öğrenci yurdu web sitesi.

## Özellikler

- Modern ve responsive tasarım
- SEO optimizasyonu
- Performans odaklı geliştirme
- Animasyonlar ve etkileşimli öğeler
- Admin paneli
- İletişim formu
- Galeri sistemi

## Teknolojiler

- Next.js 14
- TypeScript
- Tailwind CSS
- Prisma
- NextAuth.js

## Kurulum

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
